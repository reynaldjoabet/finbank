UOFS Payment Initiation API
- API version: 1.0.0
    - Generator version: 7.21.0

Uganda Open Finance Standard — Payment Initiation API (Phase 2).

Enables authorised Payment Initiation Service Providers (PISPs) to create
and manage payments from a PSU's account at an FSP. Covers domestic bank
transfers, mobile money transfers (same-MNO and cross-MNO), bill payments,
merchant payments, bulk disbursements, and scheduled payments.

Every payment follows a consent-then-execute pattern: the PISP creates a
payment consent, the PSU authorises it (via OAuth redirect, USSD, or
decoupled flow), and then the PISP submits the payment for execution.


    For more information, please visit [https://uofs.ug](https://uofs.ug)
