UOFS Account Information API
- API version: 1.0.0
    - Generator version: 7.21.0

Uganda Open Finance Standard — Account Information API (Phase 1).

Provides read-only access to account data held at Financial Service Providers (FSPs)
on behalf of Payment Service Users (PSUs) who have granted explicit consent to
authorised Account Information Service Providers (AISPs).

This specification covers banks, mobile money operators, SACCOs, and microfinance
institutions. Mobile money and SACCO/MFI extensions are included as optional
response fields controlled by consent data categories.


    For more information, please visit [https://uofs.ug](https://uofs.ug)
